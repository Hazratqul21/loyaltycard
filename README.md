# loyaltycard
LoyaltyCard — это современная экосистема и интеллектуальная платформа лояльности, предназначенная для цифровизации традиционных дисконтных систем и автоматизации взаимодействия между бизнесом и клиентами.

Проект представляет собой не просто замену пластиковым картам, а полноценный маркетинговый инструмент, использующий геолокацию, геймификацию и персонализированный подход для удержания клиентов (retention).


Основные возможности системы:
Цифровая идентификация: Безопасная авторизация через номер телефона (Firebase Phone Auth) с современным интерфейсом в стиле Glassmorphism.

Геолокационный сервис: Интерактивная карта (Google Maps), которая позволяет находить ближайшие точки продаж, прокладывать маршруты и получать уведомления при приближении к магазину.

Система уровней (Loyalty Tiers): Использование элементов геймификации. Клиенты растут в статусе (Bronze, Silver, Gold, Platinum) в зависимости от накопленных баллов, что повышает их вовлеченность.

Персонализированный контент: Лента предложений (Offers Feed), которая адаптируется под уровень лояльности конкретного пользователя.

Пуш-уведомления и Deep Linking: Система оповещений (FCM) с возможностью перехода в конкретный раздел приложения по клику на уведомление.

Реферальная программа: Встроенный механизм привлечения новых пользователей через уникальные коды приглашения.

Финансовая отчетность: Возможность экспорта истории транзакций в профессионально оформленные PDF-отчеты.
